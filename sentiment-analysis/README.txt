

Streamlit should be installed using the anaconda distribution in other to run the app.

To run the app:
* make sure you are in the same directory as of the app.py file
* open the terminal/ command prompt and set the directory to where the app.py is.
* then type ">>streamlit run app.py"


Note:

"app0.py" -> application that predcits whether a comment is negative or positive statement.

"app1.py" -> sentiment dashboard application
