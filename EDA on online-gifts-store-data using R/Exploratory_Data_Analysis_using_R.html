Exploratory Data Analysis on Online Retail Data¶

Overview:
In this notebook, I perform an exploratory data on the online retail store using R programming language. I leveraged third party library such as dplyr, readr, stringr etc. to help me perform the analysis easily and smoothly I also used the popular data visualization tool (ggplot2) to make all the visualizations and statistical plots.

Data
The data contains transaction made within a period of 1 year of countries in Europe.

Attributes:
Invoice No.: Unique Invoice Number of a customer
Stock Code : Item code
Description: Item description
Quantity: Quantity of items purchased
Invoice Date: Date of which the invoice was issued
Unit Price: unit price an item
Customer ID: unique id of a customer
Country: country of a customer where the transaction was made

--I focus my analysis on Germany only.

Analysis:

  • Sales by month
  • Sales by day name
  • Sales by day of the month
  • Top frequent Items
  • Top Items by Sales
  • Sales by holiday (I merged the holiday data with the retail data) etc.

Motive:
I did this project to showcase my skill in (exploratory data analysis) using R programming language.
